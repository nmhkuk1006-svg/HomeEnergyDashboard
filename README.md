# 🏠 Home Energy Dashboard (HED)

家庭用エネルギー管理システム

## 目的

太陽光・蓄電池・EV・電気料金を一元管理し、
電気代の最適化を支援する。

---

## システム構成

- Google Spreadsheet
- Google Apps Script
- GitHub
- ChatGPT
- （将来）Tesla API
- （将来）天気API
